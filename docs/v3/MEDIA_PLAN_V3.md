# Petit Stay — Media Plan V3

> 기준일: 2026-03-12
> 이 문서는 시각 자산(사진, 영상, 히어로 비주얼)의 규격, 소싱, 변환 규칙을 정의한다.

---

## 1. 시터 사진 규격

### 프로필 메인 사진

| 항목 | 규격 |
|---|---|
| 비율 | **3:4** (세로 우선, 모바일 카드에 최적) |
| 최소 해상도 | 600×800px |
| 권장 해상도 | 900×1200px |
| 크롭 | 얼굴 중심. 머리 위 여백 10~15%, 가슴 아래~허리 정도까지 |
| 톤 | 밝고 자연스러운 조명. 스튜디오 느낌 아닌 "진짜 사람" 느낌 |
| 배경 | 단색 또는 부드러운 실내 배경. 산만한 배경 금지 |
| 표정 | 자연스러운 미소. 정면 또는 약간 측면 |
| 복장 | 단정하고 깔끔. 로고/브랜드 노출 금지 |

### 프로필 서브 사진 (선택, 최대 5장)

| 항목 | 규격 |
|---|---|
| 비율 | **4:3** (가로) 또는 **1:1** (정사각) |
| 최소 해상도 | 800×600px |
| 내용 | 돌봄 활동 장면, 자격증, 활동 모습 (아동 얼굴은 모자이크 또는 미포함) |

### 시터 사진 금지 사항

- 선글라스, 모자로 얼굴 가림
- 과도한 보정/필터
- 셀카봉/미러 셀카
- 다른 사람(아동 포함)의 얼굴 노출 (동의 없는 경우)
- 브랜드 로고 의류
- 흡연/음주 장면

## 2. 영상 소개 규격

### 시터 자기소개 영상

| 항목 | 규격 |
|---|---|
| 길이 | **30~60초** |
| 비율 | **9:16** (세로, 모바일 전체화면) 권장. 16:9(가로)도 허용 |
| 최소 해상도 | 720p (1280×720) |
| 권장 해상도 | 1080p (1920×1080) |
| 파일 형식 | MP4 (H.264) |
| 최대 파일 크기 | 100MB |
| 오디오 | 필수. 시터 목소리가 들려야 함. 배경 음악 불필요 |

### 영상 내용 가이드

1. **자기소개** — 이름, 국적/언어, 돌봄 경력 요약 (10~15초)
2. **왜 이 일을 하는지** — 동기, 아이와의 경험 (10~15초)
3. **부모에게 한마디** — 안심시키는 메시지 (10~15초)
4. (선택) **서비스 가능 연령/활동** — 어떤 놀이, 어떤 아이 (10~15초)

### 영상 톤

- 자연스러움. 스크립트를 읽는 느낌 금지.
- 얼굴이 잘 보이는 밝은 환경.
- 배경 소음 최소화.
- 영어/일본어/한국어 등 서비스 언어로 촬영.

### 썸네일

| 항목 | 규격 |
|---|---|
| 비율 | 영상과 동일 (9:16 또는 16:9) |
| 자동 생성 | 영상 3초 시점 프레임 자동 추출 |
| 수동 업로드 | 허용. 얼굴 중심, 밝은 톤 |
| 재생 아이콘 | 플랫폼에서 오버레이 표시 |

## 3. 히어로 비주얼 방향

### 메인 랜딩 히어로

| 항목 | 방향 |
|---|---|
| 컨셉 | "여행 중인 가족이 안심하고 저녁을 즐기는 순간" |
| 구도 | 부모가 외출 준비 중, 시터가 아이와 놀아주는 장면. 또는 부모가 레스토랑에서 여유 있는 모습 + 시터 알림 수신 |
| 톤 | 따뜻한 자연광, 서울의 호텔/숙소 느낌, 다국적 가족 |
| 금지 | 스톡 사진 (iStock/Shutterstock의 전형적 가족 사진), 과도하게 연출된 스튜디오 촬영 |
| 목표 | "이 서비스는 진짜이고, 나와 비슷한 가족이 사용한다"는 느낌 |

### 런칭 전 히어로 전략

| 우선순위 | 방법 |
|---|---|
| 1순위 | 실제 파일럿 세션 촬영 (동의서 확보 후) |
| 2순위 | 시터 후보 + 스태프 자녀로 연출 촬영 |
| 3순위 | 서울 호텔 로비/객실 배경의 라이프스타일 촬영 (인물 없이 공간감) |
| 최후 수단 | 일러스트레이션 또는 추상 비주얼 (스톡 사진보다 나음) |

## 5. Cloudinary 변환 규칙

### 이미지 변환

| 용도 | 변환 규칙 | 예시 파라미터 |
|---|---|---|
| 시터 카드 (검색 결과) | 300×400, 3:4, face crop, auto quality, WebP | `c_fill,w_300,h_400,g_face,f_webp,q_auto` |
| 시터 프로필 메인 | 600×800, 3:4, face crop, auto quality, WebP | `c_fill,w_600,h_800,g_face,f_webp,q_auto` |
| 시터 프로필 서브 | 400×300, 4:3, auto crop, auto quality, WebP | `c_fill,w_400,h_300,g_auto,f_webp,q_auto` |
| 시터 아바타 (채팅/알림) | 80×80, 1:1, face crop, auto quality, WebP | `c_fill,w_80,h_80,g_face,f_webp,q_auto` |
| 히어로 비주얼 (모바일) | 750×500, auto crop, auto quality, WebP | `c_fill,w_750,h_500,g_auto,f_webp,q_auto` |
| 히어로 비주얼 (데스크톱) | 1440×600, auto crop, auto quality, WebP | `c_fill,w_1440,h_600,g_auto,f_webp,q_auto` |
| OG 이미지 | 1200×630, auto quality, PNG | `c_fill,w_1200,h_630,f_png,q_auto` |

### 영상 변환

| 용도 | 변환 규칙 |
|---|---|
| 시터 영상 재생 | Adaptive streaming (HLS), 720p + 480p |
| 썸네일 자동 추출 | `so_3,w_300,h_534,c_fill,g_face,f_jpg` (3초 시점) |

### 공통 규칙

- **원본 보존**: 업로드 시 원본은 그대로 저장, 변환은 delivery 시점에 적용
- **포맷 자동 협상**: `f_auto`로 브라우저 지원 포맷(WebP/AVIF) 자동 선택
- **레이지 로딩**: 검색 결과는 뷰포트 진입 시 로딩
- **CDN**: Cloudinary 글로벌 CDN 사용

## 6. 런칭 전 이미지 소싱 우선순위

| 우선순위 | 소싱 방법 | 사용처 | 비용 | 비고 |
|---|---|---|---|---|
| 1 | **시터 후보 실제 촬영** | 시터 프로필 사진/영상 | 저 (자체 촬영 또는 소정의 촬영비) | 온보딩 과정에서 촬영. 자연광 + 스마트폰 OK |
| 2 | **파일럿 세션 현장 촬영** | 히어로 비주얼, 마케팅 소재 | 저~중 | 동의서 필수. 아동 얼굴 미노출 또는 후면 촬영 |
| 3 | **서울 숙소 공간 촬영** | 랜딩 배경, 분위기 컷 | 중 | 호텔 파트너 협조 또는 에어비앤비 공간 |
| 4 | **일러스트레이션** | placeholder, 아이콘, 개념 설명 | 중 | 스톡 사진보다 브랜드 일관성 높음 |
| 5 | **에디토리얼 스톡** | 최후 수단. 서울 도시 풍경 등 배경용 | 저 | Unsplash/Pexels 무료. 인물 스톡 사진은 금지 |

### 아동 이미지 정책

| 규칙 | 설명 |
|---|---|
| 부모 서면 동의 필수 | 만 14세 미만 아동 이미지 사용 시 법정대리인 동의 |
| 얼굴 미노출 권장 | 후면, 실루엣, 손만 보이기, 모자이크 등 |
| 시터 프로필에 아동 사진 금지 | 시터 프로필에 돌봄 대상 아동의 얼굴이 나오면 안 됨 |
| 세션 리포트 사진 | 부모만 열람 가능, 외부 공개 금지, 보관 기간 제한 |

## 7. 브랜드 비주얼 톤

| 속성 | 방향 | 금지 |
|---|---|---|
| 색감 | 따뜻한 뉴트럴 + 신뢰의 블루/그린 포인트 | 네온, 형광, 과도한 채도 |
| 조명 | 자연광, 골든아워, 실내 따뜻한 조명 | 플래시, 형광등, 어두운 그림자 |
| 분위기 | "안심하고 맡길 수 있는 따뜻함" | "힙한 감성", "럭셔리 금장", "유치원 느낌" |
| 인물 | 다국적, 자연스러운 표정, 실제 시터/가족 | 전형적 스톡 모델, 포즈 연출 |
