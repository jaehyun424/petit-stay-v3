# Petit Stay — Copy Guide V3

> 기준일: 2026-03-12
> 이 문서는 모든 사용자 대면 텍스트의 톤, 금지 표현, 가드레일을 정의한다.

---

## 1. 공통 원칙

### 브랜드 보이스

| 속성 | 설명 |
|---|---|
| **신뢰감** | 과장하지 않는다. 사실만 말한다. 검증 가능한 것만 주장한다 |
| **따뜻함** | 부모의 불안을 이해하고, 안심시키는 톤. 차갑거나 사무적이지 않다 |
| **명확함** | 모호하지 않다. "~할 수 있습니다"보다 "~합니다". 조건은 숨기지 않고 밝힌다 |
| **간결함** | 한 문장에 하나의 정보. 긴 설명보다 짧은 확인 |

### Claims Guardrail (과장 금지)

| 금지 표현 | 이유 | 대안 |
|---|---|---|
| "한국 최초" / "세계 최초" | 검증 불가. 소개형 서비스 존재 | "한국에 표준화된 다국어 매칭 플랫폼은 부재" |
| "유일한" / "독보적인" | 경쟁사 존재 (베이비시터코리아 등) | "외국인 관광객에 특화된" |
| "완전 공백" / "아무도 없다" | 사실과 다름 | "소개형·수기형 서비스는 존재하나 표준화 플랫폼은 부재" |
| "100% 안전" / "완벽한 보장" | 과대 표현, 법적 리스크 | "검증된 시터, 안전결제, 배상책임보험으로 안심" |
| "무조건" / "반드시" / "절대" | 예외 없는 보장은 불가능 | 구체적 조건과 함께 서술 |
| "리스크 제로" / "사고 없는" | 미래 보장 불가 | "사고 예방 체계와 대응 프로토콜을 갖추고 있습니다" |
| "AI가 자동으로 매칭" | 현재 구현 아님 | "부모가 직접 검색하고 선택합니다" |
| "5억원 보험" | 미확정. 견적 전 목표치 | "배상책임보험 가입" (한도 확정 후 구체 표기) |
| "정부 인증" / "정부 공인" | 민간 등록제 시행 전, 또는 미등록 상태 | 등록 완료 후에만 사용 |

## 2. 한국어 원칙

### 톤

- **해요체** 기본. 부모/시터 대면 텍스트는 해요체.
- 운영자/법률 문서는 합니다체 허용.
- 가볍지만 가볍지 않은 — 캐주얼 앱이 아니라, 아이를 맡기는 서비스.

### 금지 표현

| 금지 | 이유 | 대안 |
|---|---|---|
| "~세요" (명령형 남발) | 지시 느낌 | "~할 수 있어요", "~해 주세요" |
| "꿀팁", "꿀시터" | 브랜드 톤과 불일치 | 사용하지 않음 |
| "찐", "갓생", 과도한 신조어 | 외국인 부모 대상이 아님 | 사용하지 않음 |
| "걱정 마세요" (단독 사용) | 근거 없는 안심은 역효과 | "~이기 때문에 안심하셔도 돼요" (근거 제시) |
| "호텔이 추천하는" | 호텔과의 관계를 오해시킬 수 있음 | "호텔에서 안내받을 수 있는" |

### 예시

| 상황 | 나쁜 예 | 좋은 예 |
|---|---|---|
| 시터 검증 설명 | "엄격한 5단계 검증으로 완벽하게 선별된 시터!" | "신원확인, 경력, 레퍼런스를 검증한 시터예요." |
| 예약 완료 | "예약이 완료되었습니다!" | "예약이 확정됐어요. Sarah가 19시에 도착할 예정이에요." |
| 보험 안내 | "걱정 마세요! 5억원 보험으로 완벽 보장!" | "세션마다 배상책임보험이 적용돼요." |

## 3. 영어 원칙

### 톤

- **Warm, clear, reassuring.** Not corporate, not cutesy.
- Active voice. Short sentences.
- Address the parent directly: "you" / "your child".
- 외국인 관광객 = 영어가 모국어가 아닐 수 있다. Simple English 우선.

### 금지 표현

| Don't | Why | Instead |
|---|---|---|
| "First-ever" / "Only in Korea" | Unverifiable. Legacy services exist | "Purpose-built for traveling families" |
| "Guaranteed safe" / "100% safe" | No absolute guarantee possible | "Verified sitters with background checks and insurance" |
| "Hassle-free" / "Worry-free" | Overused, empty promise | Be specific: "Book in 3 minutes" |
| "World-class" / "Premium" (without context) | Meaningless without specifics | Describe what makes it premium |
| "AI-powered matching" | Not implemented | "Browse and choose your sitter" |
| "Hotel-approved" / "Hotel-certified" | Misrepresents partner relationship | "Available through partner hotels" |
| "Nanny" | Implies longer-term arrangement | "Babysitter" or "sitter" |

### 예시

| Context | Bad | Good |
|---|---|---|
| Hero headline | "The Ultimate Babysitting Solution in Korea!" | "Find a verified babysitter in Seoul. Book in minutes." |
| Trust section | "We guarantee your child's 100% safety." | "Every sitter is ID-verified, reviewed, and insured." |
| CTA | "Get Started Now!!!" | "Find a sitter" |
| Session complete | "Your session has been terminated." | "Session complete. Sarah sent a care report." |

## 4. 일본어 방향

### 톤

- **です/ます体** 기본. 정중하지만 딱딱하지 않은.
- 일본 부모는 안전과 매너에 높은 기대치. "安心" "丁寧" 키워드가 자연스러운 톤.
- 일본어 여행 서비스(Konest, Creatrip JP) 톤 참고.

### 핵심 방향

- 검증 설명은 구체적으로. "身元確認済み、保育経験、レファレンス確認"
- 과장 표현 일본어 버전도 동일 적용 (「日本初」「唯一」 금지)
- 가격 표시: ¥ 환산 참고 표시 + KRW 기준
- 일본어 CS 대응 범위 확인 필요 (런칭 시 FAQ + 이메일, 채팅은 후속)

## 5. 중문(대만·홍콩) 방향

### 톤

- **繁體中文** 우선 (대만·홍콩 시장 1.5차)
- 간체중문은 2차 확장
- 정중하고 신뢰감 있는 톤. 대만 관광 서비스(KKday TW, Klook TW) 참고.

### 핵심 방향

- "安全" "驗證" "保險" 키워드 자연스럽게 배치
- 과장 표현 금지 동일 적용
- 통화: KRW 기준 + TWD/HKD 참고 환산

## 6. 법률/정책 텍스트 원칙

| 원칙 | 설명 |
|---|---|
| 두 사업계획서가 다르면 보수적 표현 선택 | GPT본이 더 보수적이면 GPT본 채택 |
| "확정"이 아닌 것은 "확정"으로 쓰지 않음 | "법률의견서로 확정 예정", "검토 목표" |
| 면책 조항은 숨기지 않고 명확하게 | 작은 글씨 금지. 별도 섹션으로 분리 |
| 제외업무는 반드시 명시 | 예약 전 동의 플로우에 포함 |
| 보험 한도는 확정값만 표기 | 견적 전: "배상책임보험 가입". 견적 후: 구체 금액 |

## 7. SEO 카피 방향

### 핵심 키워드 (검색 의도 기반)

| 언어 | 핵심 키워드 |
|---|---|
| EN | babysitter in Seoul, babysitting Seoul Korea, hotel babysitter Seoul, childcare for tourists Seoul |
| JP | ソウル ベビーシッター, 韓国 ベビーシッター, ホテル 子守り ソウル, 韓国旅行 子連れ |
| ZH-TW | 首爾 保姆, 韓國 臨時保姆, 首爾飯店 保姆服務 |

### 블로그/콘텐츠 톤

- 여행 가이드 느낌. "Seoul with Kids" 시리즈.
- 실용 정보 중심. 광고 느낌 아님.
- 내부 서비스 홍보는 자연스러운 맥락에서만.

## 8. UI 마이크로카피 원칙

| 원칙 | 예시 |
|---|---|
| 버튼은 동사로 시작 | "Find a sitter", "Book now", "시터 찾기" |
